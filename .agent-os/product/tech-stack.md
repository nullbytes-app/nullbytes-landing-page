# Technical Stack

## Frontend Technologies

**Application Framework:** Vanilla HTML5/CSS3/JavaScript
**CSS Framework:** Tailwind CSS 3.x (CDN)
**Animation Library:** GSAP (GreenSock Animation Platform) 3.12.2
**Icon Library:** Lucide Icons (latest CDN)
**Import Strategy:** CDN-based (script tags)

## Typography & Design

**Primary Font:** Orbitron (Google Fonts) - cyberpunk headers and titles
**Secondary Font:** Share Tech Mono (Google Fonts) - code/tech elements  
**Tertiary Font:** Rajdhani (Google Fonts) - body text and UI elements
**Font Provider:** Google Fonts CDN

## Visual Effects & Interactions

**Matrix Rain Effect:** Custom JavaScript implementation
**Floating Bytes System:** Custom animated particles with CSS transforms
**Glitch Effects:** CSS animations with pseudo-elements
**Interactive Background:** Custom parallax and mouse-tracking effects
**Neon Glow Effects:** CSS box-shadow and text-shadow animations

## Layout & Responsive Design

**Grid System:** CSS Grid and Flexbox
**Responsive Breakpoints:** 
- Mobile: 480px and below
- Tablet: 768px and below  
- Desktop: 1400px max-width
**Viewport Strategy:** Responsive with mobile-first approach

## Performance Optimization

**Loading Strategy:** CDN-based resources for faster initial load
**Animation Performance:** RequestAnimationFrame for smooth 60fps animations
**Image Optimization:** SVG icons and CSS-based graphics
**Code Minification:** Manual optimization of critical CSS and JS
